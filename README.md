# House to Home Real Estate Marketing Studio

Basic Next.js starter website for:

- Lifestyle image enhancement
- Virtual staging
- Feature finish close-ups
- Social media campaign packages

## Run locally

```powershell
npm install
npm run dev
```

Open http://localhost:3000

## Main pages

- /
- /services
- /packages
- /before-after
- /how-it-works
- /enquire
- /privacy
